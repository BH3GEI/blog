### 88. Merge Sorted Array
```python3
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """
        # 两个数列各一指针
        i = m - 1
        j = n - 1
        # 用于写入的辅助指针
        k = m + n - 1

        while i >= 0 and j >= 0:
            if nums1[i] > nums2[j]:
                nums1[k] = nums1[i]
                i -= 1
            else:
                nums1[k] = nums2[j]
                j -= 1
            k -= 1
        
        # 如果nums1的有效长度短，nums2剩下的部分就直接覆盖前面那部分
        while j>= 0:
            nums1[k] = nums2[j]
            j -= 1
            k -= 1
```
